2. Выписал нужные теги: 
header - Шапка сайта (может быть только один) в которой может располагаться различная информация сайта (логотип, поиск и т.д)
main - Тело нашей страницы, в main находится вся необходимая/главная информация
footer - Всегда находится внизу и может быть только один, обычно дублируют информацию которая была выше.
aside - Может иметь неограниченное количество элементов. Располагается справа/слева и имеет доп. информацию на сайте.
nav - Это набор ссылок, по которым тыкает пользователь и может перемещаться как по страницам, так и по секциям  внутри страницы
section - Нужен для того, чтоб помещать большой кусок информации, для которого не нашлось места на сайте. (неогран. кол-во)
article - Помещается логический обособленный блок (неогран. кол-во)
